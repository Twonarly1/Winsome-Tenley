## Setup
To run this project, install it locally using yarn:

terminal 1

```
$ git clone https://github.com/Twonarly1/Winsome-Tenley
$ cd Winsome-Tenley
$ yarn
$ cd Winsome-Tenley/Sanity
$ yarn
$ sanity install
$ sanity start
```
terminal 2

```
$ gcd Winsome-Tenley
$ code .
$ Add environmental variables.
$ yarn dev
```
